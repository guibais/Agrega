Perguntas Frequentes

O que é o Agrega?

O Agrega é um aplicativo com o intuito de centralizar toda informação referente a eventos acadêmicos, para que nenhum aluno perca outra oportunidade por não saber que ela estava acontecendo. Mais do que informar o aluno, temos o objetivo de ser o braço direito de todo estudante.

Posso adicionar eventos?

Sim! Contamos com você para mantermos a plataforma sempre atualizada com todos os eventos ao nosso redor. :)

E quanto custa?

É totalmente de graça! Feito por estudantes para estudantes, que assim como nós querem se desenvolver cada vez mais!
